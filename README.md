
# 填词游戏 (Tianci Game) - Crossword Clue Solver

A crossword clue answer website built with Next.js, helping users find answers to crossword puzzles quickly.

## Features

- 🔍 Smart search with fuzzy matching
- 📝 Pattern search (e.g., "I____Y")
- 📊 Browse by length, first letter, or source
- 💰 Monetization via AdSense and affiliate links
- 🚀 SEO-optimized for long-tail keywords

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Database:** PostgreSQL
- **ORM:** Prisma
- **Deployment:** Vercel

## Getting Started

This is a MVP project for crossword clue solving.
